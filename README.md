# 🔐 Caesar Cipher Security Analysis

A mini cybersecurity project demonstrating the **Caesar Cipher** and how it can be broken using **Brute Force** and **Frequency Analysis** attacks.  
This project highlights the weaknesses of classical cryptography and teaches basic cryptanalysis concepts.

---

## 📌 Features

- Caesar Cipher Encryption & Decryption
- Brute Force Attack (tries all 25 possible keys)
- Frequency Analysis Attack
- Demonstrates limitations of classical ciphers

---

## 🛠️ Technologies Used

- Python 
- ASCII character manipulation
- Basic cryptography and statistical analysis
